# Scripts Folder - Wazuh SIEM Automation

## What is this folder?

Automation scripts to deploy, test, and maintain the Wazuh SIEM project.

## Scripts in this folder

| Script | Purpose |
|--------|---------|
| deploy.sh | One-command deployment |
| test_attack.sh | Simulate SSH brute force attack |
| check_status.sh | Verify everything is working |
| view_alerts.sh | Show recent alerts in real-time |
| backup.sh | Backup custom configurations |
| unblock_ip.sh | Manually unblock an IP |

## Quick start

./deploy.sh
./test_attack.sh
./check_status.sh
