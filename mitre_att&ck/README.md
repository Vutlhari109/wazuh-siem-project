# MITRE ATT&CK Mapping Project

## What is this folder?

Complete mapping of all Wazuh custom rules to the MITRE ATT&CK framework.

## Why this matters to recruiters

MITRE ATT&CK is the industry standard for threat detection. This shows I can:
- Map detections to real attacker behaviors
- Understand TTPs (Tactics, Techniques, Procedures)
- Build detection coverage against known threats

## Files in this folder

| File | Purpose |
|------|---------|
| mapping_matrix.csv | All rules mapped to MITRE techniques |
| coverage_report.md | What attacks we can detect (and what we can't) |
| tactical_analysis.md | Detection coverage by MITRE tactic |
| technique_details.md | Deep dive on key techniques |
| detection_gaps.md | Missing detections (roadmap) |
| recruiter_summary.md | One-page for interviews |
